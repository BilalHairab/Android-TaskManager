# Task Management:

A demo application **(Android Native)** to handle and manage daily personal tasks.

# Key Technologies
- Kotlin
- Compose UI
- Material 3
- Koin (for dependency injection)
- Clean Architecture
- Coroutines
- MVI

# Versions
Built and tested using Android Studio Meerkat | 2024.3.1 on macOS.
